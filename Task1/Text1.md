# Задача
Создание инфраструктуры для реализации онлайн продаж алмазов, включая весь цикл организации.
Основные бизнес-процессы:
1. Отбор алмазов для продажи онлайн в центральном офисе.
2. Перевозка алмазов в лабораторию, их оцифровка, и возврат обратно в офис.
3. Публикация данных для покупателей, на основе которых будут приниматься решения.
4. Проведение онлайн-аукциона.
5. Отправка камней победителям торгов из офиса.

# Контекст

Изменения вызваны шоком пандемии, снижением офлайн продаж, и последующим усилением каналов онлайн продаж.

## Текущее состояние
В структуре компании существуют решения для офлайн продаж:
- ИС Алмазы - учет драгоценных камней
- ИС Аукцион - проведения офлайн продаж
- ИС «Бухгалтерия» - бухгалтерский учет и финансовые операции
- ИС «Логистика» - перевозка камней по стране и за границу
## Целевое состояние

Целевое решение должно:
- автоматизировать процессы оценки и транспортировки камней, а также подготовки и проведения онлайн аукциона
- снизить зависимость от внешних сервисов, включая размещение на собственных вычислительных мощностях
- снизить зависимость от конкретных сотрудников
- повысить безопасность обмена конфиденциальной информацией
- повысить эффективность использования ресурсов

## Ограничения

1 предполагается доработка текущей инфраструктуры компании для адаптации под цифровой канал реализации
2 для оценки предполагается использовать внешнюю ИС «Лаборатория» 
3 первый онлайн аукцион должен быть проведен через 2 месяца

# Способы решения

**Промежуточное решение**
Цель данного решения проведение первого онлайн аукциона через 2 месяцы

- подготовка цифрового аукциона без доработок ИС и интеграций, например, в Excel по утверждённым шаблонам.
- использование имеющихся информационных систем для планирования, логистики и проведения аукционов.
- заключение контракта с провайдером частного облака, чтобы там публиковать данные для клиентов.

**Целевое решение**
Цель решение - полностью автоматизированный процесс подготовки и проведения онлайн аукционов

- доработка всех существующих систем
- внедрение новой ИС "Логистика"
- перенос критически важных ИС на собственные сервера
